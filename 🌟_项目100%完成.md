## 💻 配置说明

### Vite Alias配置
所有示例的vite.config.ts都已配置别名：
```typescript
resolve: {
  alias: {
    '@ldesign/pdf-core': resolve(__dirname, '../../core/es/index.js'),
    '@ldesign/pdf-[framework]': resolve(__dirname, '../es/index.js')
  }
}
```

这确保示例能正确导入构建后的包。

### 依赖配置
所有示例的package.json都包含：
```json
{
  "dependencies": {
    "@ldesign/pdf-[package]": "workspace:*"
  }
}
```
